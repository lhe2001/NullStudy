# TeamProject
__스프링 레거시 & MVC & MyBatis & JSP & AJAX 기반 웹사이트 제작__

### 웹사이트 목적
스터디 그룹을 생성 후 멤버 소집, 스터디 멤버들과 함께 챌린지를 진행할 수 있는 사이트

### 🧭개발기간
2022.10.04 ~ 2022.10.25(약 3주간 진행)

### 👨‍👩‍👧‍👧멤버구성
**(총 다섯명)**   
애니 : 스터디그룹 관련 페이지, 메인화면   
제시 : 캘린더, 메모   
토니 : 로그인, 회원가입, 관리자   
콩 : 자유게시판   
규 : 서치   

### ⚙개발환경
JAVA 11
OS: windows 10
DB: Oracle 11g xe
WAS: Apache tomcat 9
IDE: eclipse
